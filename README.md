1. Project Overview
This project analyzes the Top 100 YouTube songs dataset (2025) to identify performance trends, audience engagement patterns, and artist dominance. The goal is to extract structured insights from raw ranking data using data cleaning, aggregation, and exploratory data analysis techniques.
The project demonstrates practical skills in data preprocessing, statistical reasoning, and visualization.
2. Objective
Analyze engagement metrics such as views, likes, comments, and ranking.
Identify top-performing artists.
Detect patterns between popularity and engagement
Compare performance distribution across songs.
Evaluate correlation between metrics.
The focus is on turning raw tabular data into interpretable insights.
